# Boronix v0.4.0 - Database DX With Drizzle

Boronix v0.4.0 adds practical database DX through Drizzle integration.

Included:

- `create-boronix --db none|sqlite|postgres`
- SQLite and Postgres DB templates
- `app/db` convention
- `boronix db generate|migrate|push|seed`
- `/notes` CRUD scaffold for DB apps
- SQLite notes example
- Doctor checks for DB projects
- Database docs and tutorial

Boronix still does not implement an ORM, migration engine, admin dashboard, auth DB adapter, or cloud deploy adapter. Drizzle owns schema and migrations.
