# Boronix v0.4.1 - Dev Error Overlay Polish

Boronix v0.4.1 refines the development error page introduced in the DX cycle.

Included:

- More polished dark modal styling.
- Subtle SVG icons for the headline, source block, diagnostics, and error banner.
- Improved readability for source frames and stack traces.
- Better spacing, borders, shadows, and mobile behavior.

No runtime API changes are included in this patch release.
